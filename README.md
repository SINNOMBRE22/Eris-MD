Eris-MD es un bot de WhatsApp versátil y potente basado en la tecnología Multi-Device. Diseñado para ofrecer una experiencia rápida, estable y llena de funciones, desde herramientas de administración de grupos hasta comandos de entretenimiento y descargas.
🚀 Características Principales
 * Multi-Device: Conexión estable sin necesidad de mantener el teléfono conectado a internet.
 * Gestión de Grupos: Comandos de ban, kick, promote, demote y configuración de bienvenida.
 * Descargas: Baja música, videos de YouTube, TikTok, Instagram y más.
 * IA Integrada: Respuestas inteligentes y generación de imágenes.
 * Convertidores: Stickers, audio a voz, video a GIF, etc.
 * Seguridad: Sistema de antiflood y antilink para mantener tus grupos limpios.
🛠️ Instalación y Despliegue
Requisitos Previos
 * Node.js (Versión 16 o superior)
 * FFmpeg (Para procesamiento de medios)
 * Libwebp (Para stickers)
Instalación Local
# Clonar el repositorio
git clone https://github.com/SINNOMBRE22/Eris-MD.git

# Entrar a la carpeta
cd Eris-MD

# Instalar dependencias
npm install

# Iniciar el bot
npm start

⚙️ Configuración (config.js)
Edita el archivo de configuración para personalizar tu bot:
 * Owner Number: Tu número de WhatsApp (con código de país).
 * Bot Name: El nombre que llevará tu instancia de Eris-MD.
 * Prefix: El símbolo para activar comandos (ej. . o !).
